# Zachary Harper — Portfolio

Personal portfolio site built with Vite + vanilla JS, using the [coughdrop](https://www.npmjs.com/package/coughdrop) CSS library.

## Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Connect the repo in [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite — no config needed (vercel.json included)
4. Deploy ✓

## Stack
- **Vite** — build tool & dev server
- **coughdrop** — custom CSS design system (bundled in `/public/coughdrop/`)
- **Vanilla JS** — scroll animations, theme toggle
- **Google Fonts** — Syne (display) + DM Mono (body)
